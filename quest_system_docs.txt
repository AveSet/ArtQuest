1. ОБЩИЙ ОБЗОР / OVERVIEW
─────────────────────────

Приложение ArtQuest содержит 1000 квестов (ещё 1 в quests.json). 
Все квесты направлены на развитие навыков рисования, анатомии, 
анимации, эффектов и сторителлинга.

Источники методологии: Marc Brunet ART School, Drawabox, Proko Anatomy,
Illusion of Life / 12 Principles of Animation, и др.

Всего повторяемых квестов (is_repeatable = true): 518
Квесты можно проходить несколько раз с обновлённым бонусом за стрик.



2. СТРУКТУРА КВЕСТА / QUEST STRUCTURE
───────────────────────────────────────

Каждый квест в JSON имеет следующие поля:

{
  "id": 1,                            // Уникальный ID (int)
  "code": "DRW-00001",                // Уникальный код (категория + номер)
  "title": {                          // Название
    "ru": "Индустриальный: линии",
    "en": "Portfolio: lighting"
  },
  "category": "drawing",              // Категория (см. раздел 4)
  "difficulty": "expert",             // Сложность (см. раздел 5)
  "description": {                    // Описание задания
    "ru": "Портфолио-уровень: мяч с ...",
    "en": "Portfolio-level: eye with ..."
  },
  "xp": 427,                          // Базовый опыт за выполнение
  "estimatedTime": 179,               // Примерное время (в минутах)
  "source": "Keys to Drawing",        // Источник/методология
  "icon": "🎨",                        // Иконка (зависит от категории)
  "color": "#6366f1",                 // Цвет (зависит от категории)
  "min_level": 32,                    // Минимальный уровень игрока
  "tags": [                           // Теги (см. раздел 6)
    "lighting", "lines", "texture",
    "expert", "drawing", "both"
  ],
  "prerequisites": [66, 232],         // ID квестов, которые нужно выполнить
  "medium": "both",                   // Формат (см. раздел 7)
  "is_repeatable": true,              // Можно ли проходить повторно
  "review_after_days": 2,             // Через сколько дней рекомендован повтор
  "streak_bonus": 1.2                 // Множитель опыта от стрика
}



3. ПОЛЯ ПОДРОБНО / FIELDS IN DETAIL
────────────────────────────────────

3.1 id
  Уникальный числовой идентификатор квеста. Используется в prerequisites
  для указания зависимостей.

3.2 code
  Строковый код в формате {КАТЕГОРИЯ}-{НОМЕР_С_ПАДДИНГОМ}.
  Примеры: DRW-00001, ANA-00003, ANM-00006, VFX-00280, STY-00286.
  Префиксы:
    DRW → drawing
    ANA → anatomy
    ANM → animation
    VFX → effects
    STY → storytelling

3.3 title
  Объект с двумя полями: ru (русский) и en (английский).
  Формат названия: "{Уровень_сложности}: {ключевой_навык}"
  Примеры: "Основы: линии", "Mastery: anatomy", "Практика: свет"

3.4 category (см. раздел 4)

3.5 difficulty (см. раздел 5)

3.6 description
  Объект ru/en с полным описанием задания.
  Для drawing: "Этюд {объект}: акцент на {аспект} и передачу материала"
  Для anatomy: "Динамическая поза: {часть_тела} в движении с акцентом на {аспект}"
  Для animation: "{тип} анимация: {объект} с принципом {принцип}, {кол-во} кадров"
  Для effects: "Эффект {тип}: базовая техника {техника}"
  Для storytelling: "Раскадровка: {N} кадров истории про {персонаж}"

3.7 xp
  Опыт за выполнение. Диапазон: 40–600.
  Распределение по сложности:
    novice:     40–65
    intermediate: 66–120
    advanced:    121–250
    master:      252–400
    expert:      407–600

3.8 estimatedTime
  Примерное время выполнения в минутах. Диапазон: 15–179.

3.9 source
  Источник методики. Возможные значения:
    "Keys to Drawing"
    "Drawabox"
    "Marc Brunet Basics"
    "Marc Brunet Figure Basics"
    "Marc Brunet Animation"
    "Proko Anatomy"
    "Morpho: Simplified Forms"
    "Figure Drawing for All It's Worth"
    "Anatomy for Sculptors"
    "Illusion of Life"
    "12 Principles of Animation"
    "Animation Mentor"
    "The Animator's Survival Kit"
    "Perspective Handbook"
    "Visual Storytelling"
    "Framed Ink"
    "Sequential Art"
    "Story: Substance, Structure, Style"
    "The Visual Story"
    "VFX Fundamentals"
    "Digital Painting Techniques"
    "Particle Systems Guide"
    "Light and Color Theory"
    "After Effects for Artists"

3.10 icon
  Эмодзи-иконка, определяется категорией:
    drawing:      🎨
    anatomy:      🦴
    animation:    🎬
    effects:      ✨
    storytelling: 📖

3.11 color
  Шестнадцатеричный цвет, определяется категорией:
    drawing:      #6366f1 (indigo)
    anatomy:      #ec4899 (pink)
    animation:    #10b981 (emerald)
    effects:      #f59e0b (amber)
    storytelling: #8b5cf6 (purple)

3.12 min_level
  Минимальный уровень игрока для доступа к квесту.
  novice:       1–4
  intermediate: 5–14
  advanced:     15–24
  master:       25–30
  expert:       31–35

3.13 tags (см. раздел 6)

3.14 prerequisites
  Массив ID квестов, которые нужно выполнить перед этим.
  Может быть пустым ([]).
  Для daily quests: если prerequisites пустой, квест может быть выбран
  в ежедневную ротацию.

3.15 medium
  Формат выполнения:
    "traditional" — традиционные материалы
    "digital"     — цифровые инструменты
    "both"        — любой формат

3.16 is_repeatable
  Можно ли проходить квест повторно. 
  Повторяемые квесты: 518 шт. (обычно novice/intermediate, study/practice).
  Неповторяемые: advanced+ квесты, мастерские работы, портфолио.

3.17 review_after_days
  Через сколько дней рекомендован повтор.

3.18 streak_bonus
  Множитель опыта от стрика. Диапазон: 1.0–1.5.
  Чем выше множитель, тем выгоднее проходить квест при длинном стрике.



4. КАТЕГОРИИ / CATEGORIES
──────────────────────────

4.1 drawing (Рисование)
  Иконка: 🎨  Цвет: #6366f1
  Всего квестов в файле: ~400
  Описание: базовые навыки рисования — линии, формы, перспектива,
  композиция, светотень, работа с цветом и материалами.
  
  Разделы (по тегам):
    • lines — линии и контроль
    • form — объём и форма
    • perspective — перспектива (1-2-3 точки схода)
    • composition — композиция кадра
    • shading — штриховка и затенение
    • lighting — свет и тень
    • texture — текстуры и материалы
    • color — цвет и колористика
    • construction — конструктивный рисунок
    • warmup — разминка
    • practice — практика
    • workflow — рабочий процесс
    • depth — глубина
    • space — пространство
    • value — тональность
    • control — контроль линии
    • digital — цифровое рисование
    • traditional — традиционное рисование

4.2 anatomy (Анатомия)
  Иконка: 🦴  Цвет: #ec4899
  Всего квестов в файле: ~250
  Описание: анатомия человека — скелет, мышцы, пропорции,
  жесты, динамические позы.
  
  Разделы:
    • skeleton — скелет

    • gesture — жестовый рисунок
    • proportions — пропорции

4.3 animation (Анимация)

  Описание: принципы анимации по Диснею — timing, spacing,
  squash & stretch, arcs, secondary action, и др.
  
  Разделы (основаны на 12 принципах анимации):

    • anticipation — подготовка к действию

    • slow-in-out — замедление входа/выхода
    • arcs — дуги движения

    • spacing — интервалы

    • walk-cycle — цикл ходьбы

7. МЕДИУМ / MEDIUM
───────────────────

"digital"      — задание выполняется в цифровом формате (графический
                 планшет, iPad, ПК)
"traditional"  — задание выполняется в традиционных материалах
                 (бумага, карандаш, маркеры, краски)
"both"         — задание можно выполнить в любом формате



8. ЕЖЕДНЕВНЫЕ КВЕСТЫ / DAILY QUESTS
─────────────────────────────────────

Механика:
  • Каждый день генерируется 3 ежедневных квеста
  • Отбор происходит детерминированно на основе seed (дата + уровень)
  • В ротацию попадают квесты с is_repeatable = true (518 шт.)
  • Гарантированный состав: 1 drawing + 1 animation + 1 из остальных
  • Выбор взвешен по difficulty с учётом уровня игрока

Проверка:
  • При загрузке приложения проверяется lastDailyQuestDate
  • Если дата изменилась — запускается generateDailyQuests()
  • Если квест выполнен сегодня — он отмечается в completedToday
  • Готовые daily-квесты заменяются новыми на следующий день

12. ОСОБЕННОСТИ И НЮАНСЫ / FEATURES & NUANCES
────────────────────────────────────────────────

12.1 Повторяемость
  • 518 из 1000 квестов — повторяемые
  • Все novice — повторяемые (для ежедневной практики)
  • Большинство intermediate — повторяемые
  • Почти все advanced, master, expert — НЕ повторяемые
  • Исключение: expert-квесты с пометкой "Portfolio" могут быть
    повторяемыми (для переработки портфолио)

12.2 Пререквизиты
  • Квесты могут требовать выполнения других квестов
  • Пререквизиты указываются массивом ID: [66, 232]
  • dailyQuests выбираются только из квестов без пререквизитов
  • Для мастер-квеста могут требоваться 2+ intermediate квеста

12.3 Бонус стрика (streak_bonus)
  • Множитель опыта за непрерывное выполнение
  • Диапазон: 1.0 (нейтрально) — 1.5 (максимальный бонус)
  • Выше множитель у коротких практических квестов
  • Ниже у портфолио-работ

12.4 Время на повтор (review_after_days)
  • 0 — не требуется повтор
  • 2–7 — интенсивный повтор (для базовых навыков)
  • 14 — умеренный повтор
  • 30 — редкий повтор

12.5 Распределение по сложности (приблизительно):
  • novice:      ~300 квестов
  • intermediate: ~250 квестов
  • advanced:    ~250 квестов
  • master:      ~120 квестов
  • expert:      ~80 квестов

12.6 Распределение по категориям:
  • drawing:      ~400 квестов
  • anatomy:      ~250 квестов
  • animation:    ~180 квестов
  • effects:      ~100 квестов
  • storytelling: ~70 квестов
